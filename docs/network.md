# 个人理解

    所有服务都是基于C/S或B/S

    通信问题大分类：
        1、客户端请求没有到达服务端，这是一类网络错误

        2、客户端请求到达服务端，服务端如何处理，这是一类分类，例如500，400，300，200，100等等

        3、服务端返回200的情况，业务有自己的逻辑处理code，还是沿用http状态码？
